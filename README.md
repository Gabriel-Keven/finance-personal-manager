## 📖 Sobre o projeto

O **Finance Personal Manager** é uma aplicação web desenvolvida para auxiliar no gerenciamento de finanças pessoais.

O sistema permite organizar receitas e despesas, facilitando o acompanhamento financeiro e proporcionando uma visão mais clara da situação financeira do usuário.

Este projeto está sendo desenvolvido com foco em boas práticas de arquitetura, separação entre frontend e backend e utilização de tecnologias modernas.

## ⚙️ Como executar

### Backend

Entre na pasta:

```bash
cd backend-quarkus
```

Execute:

```bash
./mvnw quarkus:dev
```

ou

```bash
mvn quarkus:dev
```

O backend ficará disponível em:

```
http://localhost:8080
```

---

### Frontend

Entre na pasta:

```bash
cd frontend-angular
```

Instale as dependências:

```bash
npm install
```

Execute:

```bash
npm start
```

ou

```bash
ng serve
```

A aplicação ficará disponível em:

```
http://localhost:4200
```
